Objetivo:
Seguir alguma das seguintes arquiteturas: MVC, MVVM, MVP, Flux ou Redux;
Adicionar itens (produtos, pets, pessoas) em uma lista em memória.
Requisitos Mínimos:
Haver uma entidade para o item com ao menos identificador e nome/descrição;
Um Service/Business que valida se o nome do novo item tem mais de 2
caracteres e se não há itens com o mesmo nome na lista;
Uma View que exibe um input, um botão de adicionar e uma lista de itens.
